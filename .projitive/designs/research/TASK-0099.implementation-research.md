## Code Architecture and Implementation Findings

### 模块结构 (source/index.ts#L1-L30)
Airx 的核心模块：
- `app/` - 应用生命周期管理 (createApp, mount)
- `element/` - DOM 元素创建与操作
- `render/` - 渲染引擎与插件系统
- `signal/` - Signal 响应式原语集成 (基于 signal-polyfill)
- `symbol/` - 内置组件符号 (Fragment, Portal 等)
- `types/` - TypeScript 类型定义
- `logger/` - 日志系统

### 构建产物
- ESM: `.js` + `.d.ts`
- 类型声明: `.d.ts`
- 无 CJS 特殊处理，通过 ESM 兼容

airouter 有 `designs/core/architecture.md` 和 `designs/core/style-guide.md`，airx 参照此结构创建了对应文档。
