## Code Architecture and Implementation Findings

### Current State (0.8.0)
- **Core**: JSX + Signal-driven reactivity, no runtime dependencies
- **Rendering**: Browser DOM + SSR (string-based, no streaming)
- **Bundle**: Zero dependencies, lightweight focus
- **Testing**: 378 tests, vitest + jsdom

### Project Structure
- TypeScript source only (no bundled output in repo)
- Build via `tsc` to output/ directory
- No CI/CD that publishes npm (manual publish process indicated by version sync issue)

### Known Gaps (from architecture.md "Out of Scope")
- No built-in routing → delegated to airx-router
- No built-in styling
- No server-side data fetching
- No concurrent SSR
- No production SSR streaming
