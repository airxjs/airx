## Code Architecture Findings

### API Surface (source/index.ts)
```typescript
// 公开 API
export type { AirxApp, Plugin }
export { createApp }
export { Fragment, component, createElement }
export type { AirxComponent, AirxElement, AirxChildren, AirxComponentContext }
export { inject, provide, onMounted, onUnmounted }
export { createRef, watch }
export type { Ref }
```

### Key Implementation Files
- `source/app/app.ts` — App 根容器
- `source/element/element.ts` — JSX 元素创建
- `source/signal/index.ts` — Signal 实现（对齐 TC39）
- `source/render/basic/` — provide/inject、生命周期 hooks
- `source/types/types.ts` — JSX 全局类型扩展

### JSX 类型扩展（阻塞发布）
`source/types/types.ts` 中 `IntrinsicElements` 等接口需要维护 JSX 类型扩展，这是 beta API 的一部分。

## Release Readiness Assessment

### Must Fix Before Release

1. **TypeScript 类型检查缺失**
   - `package.json` 无 `typecheck` 脚本
   - 需要添加并验证 `tsc --noEmit` 通过

2. **CHANGELOG.md 缺失**
   - 需要记录 0.4.0 → 0.6.0 的所有变更

3. **npm publish dry-run 验证**
   - 运行 `npm publish --dry-run` 确认文件列表

4. **git tag v0.6.0**
   - 合并 dev → main 后打 tag

### Recommended for Stable Release

1. **升级 README.md**
   - 记录 0.6.0 的新 API（Signal 对齐、Fragment 等）

2. **下游通知**
   - `airx-router` peerDeps 更新
   - `vite-plugin-airx` 同步

## Implementation Plan

### Phase 1: Pre-flight Checks
1. 添加 `typecheck` 脚本到 `package.json`
2. 运行 `tsc --noEmit` 确认无类型错误
3. 运行 `npm run lint` 确认干净
4. 运行 `npm test` 确认全通过
5. 运行 `npm publish --dry-run` 验证包内容

### Phase 2: Documentation
1. 创建 `CHANGELOG.md` 记录 0.5.0 / 0.6.0 变更
2. 更新 `README.md` 文档（如需要）

### Phase 3: Release
1. 更新 `package.json` 版本 `0.6.0-beta.1` → `0.6.0`
2. git commit 版本更新
3. git tag `v0.6.0`
4. 合并 dev → main
5. 推送 tags
6. 触发 `publish.yml` 发布 npm

### Phase 4: Downstream Notification
1. 确认 airx-router peerDeps 需要更新
2. 确认 vite-plugin-airx 需要更新
3. 通知下游或创建对应 issue/PR

## Risks and Open Questions

### Risk 1: TypeScript 类型覆盖空白
- 当前无 `typecheck` 脚本，未验证全局类型正确性
- **缓解**：立即添加并运行 `tsc --noEmit`

### Risk 2: API 破坏性变更未知
- 0.4.0 → 0.6.0 跨度较大（2 minor 版本）
- **缓解**：检查 `source/index.ts` 公开 API 数量，对比 api-compatibility-matrix.md

### Risk 3: 下游阻塞
- airx-router 和 vite-plugin-airx 依赖本版本
- **缓解**：发布后立即通知下游

### Open Question
- 0.5.0 是否存在？版本号跳跃是否有意为之？
- 需要确认是否有跳过的版本需要记录
